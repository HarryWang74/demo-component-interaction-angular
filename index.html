Components Interaction

componant 传数据到另一个 componant

  • parent/child

    • 从 parent 向 child 传数据，使用 input
      • Child 使用 input 从 parent 接收 object 并显示

  • no parent/child

    • Angular document example
      • 点击按钮，从任务列表取任务，发布给下面的宇航员列表
      • 宇航员列表中，可以确认每个宇航员是否接受任务，一旦接受无法更改
      • History 列表记录所有操作
    • My example
      • 在一个 component 中点击按钮，通过 service 发送 object 到另一个 component

componant 调用另一个 componant 的 method

  • parent/child

  • no parent/child

    • My example
      • 在一个 component 中点击按钮，通过 service 调用另一个 component 里的 method
      • 需要再调用 method 时传数据，参考传数据 example. 实际上两种写法是一样的，都通过观察 service 里 string 的改变实现。
